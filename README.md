# bits

Testing Bit Manipulation and various other small intrinsics

* Add overflow
* Sub underflow
* Is Power of 2
* Round Power of 2
* Pop count
* Count leading zeros
* Count trailing zeros

Multiplatform wrappers for Windows, Linux, OS X, iOS, Android

Accelerated popcount, clz, ctz for uint16_t, uint32_t and uint64_t with unaccelerated software fallbacks
